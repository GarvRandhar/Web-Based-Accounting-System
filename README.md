## Modern Web-Based Accounting for Real Finance Workflows
Accounting Pro is a modern, production-ready accounting system built to manage core business finance workflows in one place. It brings together a polished dashboard, ledger management, chart of accounts, invoices, bills, customers, vendors, products, warehouses, stock tracking, tax groups, cost centers, currencies, employees, payroll, and fixed assets in a clean, dark-themed interface.

The project has been fully updated and is live for demo at the link below.

## Live Demo
Experience the deployed application here:

https://web-based-accounting-system.onrender.com

## Highlights
- Beautiful dark-mode dashboard with key financial summaries
- Real-time style overview cards for assets, liabilities, equity, and YTD income
- Revenue vs. expenses charting and expense breakdown visualization
- General ledger and chart of accounts management
- Invoices, bills, customers, vendors, and product workflows
- Warehouse, stock ledger, and fixed asset modules
- Employee, payroll, cost center, currency, and tax group support
- Built for a smooth accounting operations experience

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/GarvRandhar/Web-Based-Accounting-System.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Web-Based-Accounting-System
   ```
3. Install dependencies according to the project setup.
4. Run the application using the appropriate start command for this codebase.
