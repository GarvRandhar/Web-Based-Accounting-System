## Overview
Accounting Pro is a modern, production-ready web-based accounting system for managing core finance workflows in one place. It includes a polished dashboard, journal entries, chart of accounts, invoices, bills, customers, vendors, products, warehouses, stock ledger, tax groups, cost centers, currencies, employees, payroll, and fixed assets.

The application has been updated completely and is now deployed as a live demo.

## Live Demo
Try the live application here:

https://web-based-accounting-system.onrender.com

## Key Features
- Modern dark-mode accounting dashboard
- Total assets, liabilities, equity, and YTD income summary cards
- Revenue vs. expenses analytics
- Expense breakdown visualization
- Recent invoices and bills-to-pay overview
- General ledger and chart of accounts management
- Multi-module accounting workflow for day-to-day operations

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/GarvRandhar/Web-Based-Accounting-System.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Web-Based-Accounting-System
   ```
3. Install dependencies according to the project setup.
4. Run the application using the appropriate start command for this codebase.
